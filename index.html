Privacy Policy for 易联AI建联助手

Last updated: July 6, 2026

易联AI建联助手 respects your privacy. This extension does not collect, sell, share, or disclose personal or sensitive user data to the developer or to any third party, except where data is processed locally in the user's browser to provide the extension's core features.

1. Data Collection

This extension does not collect personal information such as names, email addresses, passwords, payment information, authentication information, precise location, or communications content.

2. Data Access and Use

The extension may access webpage content, selected text, or user-provided input only when needed to provide AI-assisted connection, communication, drafting, organization, or productivity features requested by the user. This data is used only to provide the extension's functionality.

3. Local Processing

Extension settings and temporary working data are stored locally in the user's browser when necessary. The developer does not use this data for advertising, profiling, creditworthiness, or any unrelated purpose.

4. Remote Services

If the user chooses to use AI-related features that require a remote service, the minimum necessary user-provided content may be sent securely to the relevant AI service solely to generate the requested result. The extension does not sell this data or share it with unrelated third parties.

5. Data Sharing

The extension does not sell, rent, or transfer user data to third parties for advertising, analytics, or marketing purposes. User data is shared only when necessary to provide a user-requested feature, comply with applicable law, or protect users and the service from abuse or security risks.

6. Data Retention

The extension does not maintain a developer-controlled database of personal user data. Locally stored settings remain on the user's device and can be removed by uninstalling the extension or clearing browser extension data.

7. Security

Reasonable safeguards are used to protect data handled by the extension. No method of transmission or storage is completely secure, but the extension is designed to minimize data access and data sharing.

8. Children

This extension is not directed to children and does not knowingly collect data from children.

9. Changes to This Policy

If the extension's data practices change, this privacy policy will be updated before the changed practices are released.

10. Contact

If you have questions about this privacy policy, please contact the developer through the support contact information provided on the Chrome Web Store listing.
