Last updated: July 8, 2026

This page is the official privacy policy for the Chrome extension named 易联AI建联助手. It describes what data the extension accesses, how that data is used, and how user privacy is protected.

1. Data Collection

易联AI建联助手 does not sell user data and does not use user data for advertising, profiling, creditworthiness, or unrelated purposes.

The extension may access webpage content, selected text, or user-provided input only when the user actively uses features of the extension, such as AI-assisted communication, drafting, organization, connection management, or productivity features.

2. Purpose of Data Use

Any accessed data is used only to provide the user-requested functionality of the extension. The extension does not use this data for unrelated business purposes, advertising, or tracking.

3. Local Data

Extension settings and temporary working data may be stored locally in the user's browser when necessary for the extension to function. Locally stored data remains on the user's device and can be removed by uninstalling the extension or clearing browser extension data.

4. Remote Processing

If the user chooses to use AI-related features that require a remote service, the minimum necessary user-provided content may be sent securely to the relevant service solely to generate the requested result. The extension does not sell this data or share it with unrelated third parties.

5. Data Sharing

User data is not sold, rented, or transferred to third parties for advertising, analytics, or marketing. Data is shared only when necessary to provide a user-requested feature, comply with applicable law, or protect users and the service from abuse or security risks.

6. Data Retention

The developer does not maintain a developer-controlled database of personal user data collected by the extension. Temporary data is retained only as long as necessary to provide the requested functionality.

7. Security

Reasonable safeguards are used to protect data handled by the extension. The extension is designed to minimize data access and data sharing.

8. Children

This extension is not directed to children and does not knowingly collect data from children.

9. Changes to This Privacy Policy

If the extension's data practices change, this privacy policy will be updated before the changed practices are released.

10. Contact
