Privacy Policy for RPA 自动化助手

This page is the official privacy policy for the Chrome extension named “RPA 自动化助手.” It explains how the extension accesses, uses, stores, and protects user data. This page is a dedicated privacy policy, not a product homepage or promotional website.

1. Scope

This policy applies only to the RPA 自动化助手 Chrome extension. The extension helps users perform automation tasks on webpages they actively select, such as identifying page elements, clicking, entering information, reading task results, and saving automation task configurations.

2. Data Accessed by the Extension

To complete an automation task initiated by the user, the extension may access the following data:

  • Webpage content, page elements, and task results relevant to the active task;
  • Automation instructions, form content, and task configurations entered by the user;
  • Extension settings, task templates, and temporary task execution state.

The extension does not access this data for advertising, profiling, creditworthiness, or purposes unrelated to its core functionality. It does not intentionally collect passwords, payment information, or precise location data unless the user explicitly instructs the extension to process the relevant webpage fields as part of an automation task.

3. How Data Is Used

Data is used only to perform automation operations explicitly requested by the user, display task results, save user settings, and restore task state. The developer does not use user data for advertising, cross-site tracking, data sales, or any purpose unrelated to the extension's core functionality.

4. Local Storage and Retention

Extension settings, task templates, and necessary temporary data may be stored in the browser's extension storage. The developer does not maintain a database for the long-term storage of personal user data. Local data is retained until the user deletes the relevant configuration, clears the extension's data, or uninstalls the extension. Temporary execution data is retained only for as long as needed to complete the relevant task.

5. Data Transmission and Sharing

By default, automation task data is processed on the user's device and in the user's browser. It is not sent to the developer or third parties for advertising or marketing. If the user actively uses a feature that requires an external API or third-party service, the extension transmits only the data necessary to complete that request to the service selected or configured by the user. Such data is subject to the privacy policy of the applicable third-party service.

The developer does not sell or rent user data and does not share it with unrelated third parties. Limited processing or disclosure may occur only when necessary to provide a feature requested by the user, comply with applicable law, or protect users and the service from abuse or security risks.

6. Browser Permissions

Permissions for webpage access, script execution, and storage are used only to perform automation tasks on webpages selected by the user, read task results, and save extension settings. These permissions are not used for activities unrelated to the extension's stated functionality.

7. User Control and Data Deletion

Users may delete saved tasks and settings at any time. Users may also delete locally stored data by clearing the extension's data or uninstalling RPA 自动化助手. Stopping a task or denying access to a particular website prevents the extension from continuing to process data from that website.

8. Data Security

The extension uses reasonable technical measures to limit the scope of data access and processes data only for the time and purpose required to provide the requested functionality. No method of storage or transmission can be guaranteed to be completely secure, but the extension is designed to minimize data access, retention, and sharing.

9. Children's Privacy

This extension is not directed to children, and the developer does not knowingly collect personal data from children.

10. Changes to This Policy

If the extension's functionality or data practices change, this policy will be updated before the relevant changes are released. The updated date will be displayed on this page.

11. Contact

For questions about this privacy policy or the extension's data practices, contact the developer, maqitong, through GitHub Issues.
